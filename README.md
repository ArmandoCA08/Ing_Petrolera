# Ingeniería Petrolera
Repositorio académico y técnico enfocado en el desarrollo, organización y documentación de proyectos relacionados con la **Ingeniería Petrolera**, incluyendo herramientas computacionales, análisis de datos, programación aplicada y material de apoyo para distintas áreas de la disciplina.

## Descripción
Este repositorio tiene como objetivo integrar proyectos, códigos, documentos y recursos desarrollados para apoyar el aprendizaje y la aplicación de la programación en temas de Ingeniería Petrolera.
El contenido se irá actualizando progresivamente, incorporando trabajos relacionados con áreas como:

- Perforación de pozos.
- Producción de hidrocarburos.
- Yacimientos.
- Análisis nodal.
- Flujo multifásico.
- Automatización de cálculos.
- Visualización y análisis de datos.
- Herramientas computacionales aplicadas a la ingeniería petrolera.
